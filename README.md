# Instructions: 

You will need a .env file formatted as follows :  
DISCORD_TOKEN= <DISCORD_TOKEN>  
GROQ_API_KEY= <GROQ_API_KEY>  

You will also need to provide your own prompt.txt file to determine the behavior of the AI.  
  
## Groq LLM  
Go here to get a Groq API key : (https://console.groq.com/keys)  

## Discord Token  

To get a token, go to discord.com/developers/applications and make an application. Name it what you want your bot to be called.  
Watch this video for more information : https://www.youtube.com/watch?v=YD_N6Ffoojw&t=1481s
